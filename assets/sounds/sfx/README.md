# SE 差し替え方法

各フォルダに `.wav` `.mp3` `.ogg` `.m4a` のいずれかを **1個** 入れてください。
ファイル名はなんでもOK。複数入れた場合は最初の1個が採用されます。

完了したら、プロジェクト直下の `update-sfx.command` をダブルクリックしてください。
マニフェストが更新され、ブラウザでリロードすれば差し替え音が鳴ります。

ファイルが入っていないフォルダは、これまで通り合成音（Web Audio API）が鳴ります。

## フォルダと用途

| フォルダ | いつ鳴るか |
|---|---|
| `eat-great/`     | おすすめネタを食べた時（コンボ加算と同時） |
| `eat-mid/`       | おすすめ以外のネタを食べた時 |
| `chew/`          | 食べた瞬間（毎回。上の音に重なる） |
| `gulp/`          | お茶を飲む（Z）|
| `gari/`          | ガリを食べる（E）|
| `register/`      | 会計画面：合計金額確定の瞬間 |
| `coin/`          | 会計画面：1皿ごとの加算（連続で鳴る） |
| `buzz/`          | エラー音（操作不可、ガリ切れ、イライラ閾値突破）|
| `ban/`           | リザルト画面：各表示段階の打撃音（5回）|
| `jingle/`        | お勘定演出 → 会計画面に切り替わる時 |
| `bill-intro/`    | 「お勘定」演出開始時（Q押下 / 強制会計）|
| `score-fanfare/` | リザルト：最終スコア大写し |

詳しくは `se-report.md` を参照。
